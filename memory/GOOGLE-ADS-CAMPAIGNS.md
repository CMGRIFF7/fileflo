## Conversion Tracking — Set Up Before Launch

1. Go to Google Ads → Tools → Conversions
2. Create conversion: "Trial Signup"
   - Source: Website
   - Category: Sign-up
   - Count: One
   - Lookback window: 30 days
3. Install on the /thank-you or post-signup confirmation page at getfileflo.com
4. Also track: "Contact Form Submit", "Demo Request"
